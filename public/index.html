Xget

High-performance acceleration engine for developer resources built on Cloudflare Pages.

Features

  • Multi-platform support (40+ platforms: GitHub, npm, PyPI, Docker Hub, etc.)
  • Smart caching with protocol-aware strategies
  • HTTP Range request support for partial downloads
  • Container registry authentication proxy
  • Retry logic with exponential backoff
  • Comprehensive security headers
